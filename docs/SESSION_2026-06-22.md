# Printo — Development Session Log (2026-06-22)

A full record of one working session that took **Printo** from a bare extraction
POC to a polished, secured, gateway-backed application. Stack: **FastAPI**
(backend) + **Streamlit** (frontend) + **SQLite**, for Coral Business Solutions /
RealSoft ERP.

> Secrets (VPS root password, gateway API key) are **not** included here — they live
> only in the gitignored `.env`. This log is safe to commit.

---

## 1. What Printo does

```
Login → Upload drawing → preprocess (render/clean/crop) → prepass (PDF text / OCR)
      → AI extraction (gateway) → validate (18 rules) → map to RealSoft ERP → live report
```

Everything streams live to the UI via Server-Sent Events. Designed to run with **no
API key in the app** — all inference is routed through an AI gateway, with mock
fallback for offline/demo.

## 5. How to run

```bash
pip install -r requirements.txt
# backend (run from backend/ — bare imports require that cwd)
cd backend && uvicorn main:app --port 8000
# frontend (from repo root)
streamlit run frontend/app.py --server.port 8501
```
Open http://localhost:8501 → sign in **Admin / Admin@123**.

**Gateway (live mode):** open the SSH tunnel so `localhost:8095` reaches the VPS gateway:
```
ssh -i ~/.ssh/hostinger_ssh_nopass -N -L 8095:127.0.0.1:8095 root@76.13.244.21
```
With `AI_PROVIDER=auto` the app uses the gateway when reachable, else sidecar/mock.
Verify: `python tools/gateway_smoketest.py`.

## 7. Known issues / next steps

- **Gateway latency** ~60s/drawing (two sequential Claude-CLI calls: extract + ERP_MAP).
  Timeouts raised; could parallelize or use local mapper for speed.
- **Gateway `ERP_MAP` is sparse** (~3 fields in testing) — enrich its template
  (`/opt/printo-ai-gateway/src/templates.ts` on the VPS); local `realsoft_mapper` is richer.
- **Tunnel is a background process** — dies on reboot (app then falls back to mock).
  For permanent, tunnel-free production: issue a valid TLS cert for
  `https://printo-gw.hopetech.me` (DNS + certbot) — its cert is currently a hostname
  mismatch.
- **Remember-me** extends token TTL only; true cross-refresh persistence needs cookie
  storage (Streamlit resets session on hard refresh).
- Pending from client/Coral: real RealSoft API spec + credentials; real sample drawings
  for accuracy numbers.
